# 🎯 Motivational Poster Website

**Day 44 / 100 — #100DaysOfCode**

This project is a **Motivational Poster Website** built using **HTML and CSS**.  
The goal of this project is to create a clean motivational poster layout combining an image with an inspiring quote.

The poster highlights the idea that **consistent effort leads to meaningful progress**.

---

# 📌 Project Overview

This webpage displays:

- A developer workspace image
- A motivational quote
- A centered poster layout
- A dark cinematic theme

The design follows the classic poster format:

```
Image
↓
Title
↓
Subtitle
```

---

# 🖼 Preview

The poster contains the message:

> **Consistency Beats Motivation**  
> Small progress every day creates big results.

# 💡 Fun Fact

Websites are usually built using **three main technologies**:

- **HTML** → Structure  
- **CSS** → Design  
- **JavaScript** → Interactivity  

Unlike languages like **C or C++**, which compile into a single executable program, web applications combine multiple files that the **browser renders together**.
